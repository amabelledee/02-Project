# Bookgiggidy

## Overview
Take the hassle out of organizing events with Bookgiggidy!
Bookgiggidy is an entertainment app that helps event coordinators
manage all their events, vendors, artists, and budget, all in one place.

##Technologies Used
* Followed MVC Design
* JavaScript
* JQuery
* Google Calendar
* Node.js
* MySql
* Bootstrap
* Express
* Handlebars
